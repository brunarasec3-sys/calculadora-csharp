# 🧮 Calculadora em C#

Projeto simples de calculadora desenvolvido em C#.

---

## 🚀 Funcionalidades

- ✔ Soma
- ✔ Subtração
- ✔ Multiplicação
- ✔ Divisão
- ✔ Validação de entrada com TryParse
- ✔ Tratamento de divisão por zero
- ✔ Menu interativo no console
- ✔ Repetição de operações

---

## 💻 Como funciona

O usuário escolhe uma operação pelo menu, digita dois números e o sistema calcula o resultado, tratando possíveis erros.

---

## 🧠 Aprendizados

Este projeto foi importante para praticar:

- Estruturas de decisão (`if/else`)
- Laços de repetição (`while`)
- Validação de dados (`TryParse`)
- Controle de fluxo (`continue`)
- Organização de código
